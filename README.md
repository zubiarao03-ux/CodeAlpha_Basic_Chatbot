# 🤖 Basic Chatbot

## Project Description

The Basic Chatbot is a simple rule-based chatbot developed using Python as part of the **CodeAlpha Python Programming Internship**.

The chatbot interacts with users using predefined responses and continues the conversation until the user types **bye**.

---

## Features

- Greets the user
- Answers common questions
- Responds using predefined rules
- Runs continuously until the user exits
- Beginner-friendly Python project

---

## Technologies Used

- Python 3
- Visual Studio Code

---

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

```bash
python chatbot.py
```

4. Start chatting.
5. Type **bye** to exit.

---

## Sample Questions

- hello
- how are you
- what is your name
- who made you
- bye
